# 記憶體即算力：Groq LPU 與存算一體的賽道分野

> 從馮紐曼瓶頸到後摩爾時代的兩條突圍路徑 · April 2026
> 涵蓋主題：Groq LPU 架構、存算一體（IMC）、知存科技、NVIDIA 收購策略、AI 推論市場

> 📌 **後記預告：** 本系列將在後續文章中完整介紹「杭州九小龍」——中國 AI 新創浪潮的九家核心企業，敬請期待。

## 一、共同的敵人：記憶體牆（Memory Wall）

現代 AI 算力的最大瓶頸，並非電晶體數量不夠，而是**記憶體與運算單元之間的數據搬移代價**。

傳統 GPU 架構（如 NVIDIA H100）的模型權重存放在外部 HBM（高頻寬記憶體）堆疊中，每次推論都需要將數十億個參數反覆搬入算力核心，消耗巨量能源與時間。這就是「記憶體牆」——它正在成為 AI 算力擴張的天花板。

面對同一個問題，業界出現了兩條截然不同的突圍路徑：

- **Groq LPU（SRAM 路線）**：把記憶體搬到晶片上，縮短搬移距離
- **存算一體 IMC（Flash/ReRAM 路線）**：讓記憶體單元本身直接做計算，根本不搬移

---

## 二、Groq LPU：NVIDIA 花 200 億美元買下的答案

### 架構原理

Groq 的 LPU（Language Processing Unit）是一種**全數位、靜態排程**的推論加速器。其核心創新在於：

- 將模型權重存放於**片上 SRAM**，而非外部 HBM
- 編譯器在執行前**預先排定所有運算順序**，消除 pipeline stall 與 cache miss
- 每個時脈週期都是確定性的，延遲極低且可預測

### 關鍵規格對比

| 規格 | Groq 3 LPU | NVIDIA Rubin GPU |
|------|-----------|-----------------|
| 記憶體類型 | 片上 SRAM | 外部 HBM |
| 記憶體容量 | ~500 MB | 288 GB |
| 記憶體頻寬 | **150 TB/s** | 22 TB/s |
| 精度支援 | FP8 / INT8 | FP32 / BF16 / FP8 |
| 適用場景 | 低延遲推論 | 訓練 + 大批次推論 |
| 功耗等級 | 中低（~100–300W） | 極高（700W+） |

頻寬差距說明一切：Groq 以不到 GPU 1/500 的容量，換來**近 7 倍的記憶體頻寬**。對於聊天機器人、即時 Agent 等每次只處理一個 token 串的場景，這個取捨完全合理。

### NVIDIA 為何收購

2025 年底，NVIDIA 以 200 億美元收購 Groq 推論業務單位，是其歷史上最大宗收購。戰略邏輯清晰：

- AI 市場正從**訓練主導**轉向**推論主導**——預計 2030 年推論佔 AI 算力支出的 75%
- HBM 供應鏈持續緊張，SRAM 路線提供戰略備援
- 整合進「NVIDIA AI Factory」架構，補足低延遲推論的缺口

## 三、存算一體（IMC）：知存科技的另一條路

### 架構原理

存算一體走的是更激進的物理路徑。

傳統晶片的邏輯：**存儲 → 搬移 → 計算**

IMC 的邏輯：**存儲 = 計算**

利用歐姆定律（電壓 × 電導 = 電流），Flash 或 ReRAM 記憶體陣列本身就能完成矩陣乘法。一次記憶體讀取操作，即可完成百萬級參數的乘加運算。同等工作量下，傳統 GPU 需要超過 5 萬次記憶體存取。

### 知存科技（WITINMEM）

杭州知存科技成立於 2017 年，是**全球最早將存算一體晶片商業化量產**的企業。

- **2016 年**：首席科學家完成國際首個 Flash 存算一體深度學習晶片驗證
- **2019 年**：發布國際首個存算一體產品 WTM1001
- **2022 年**：量產 WTM2101，進入 AR 眼鏡、助聽器、TWS 耳機等消費品
- **WTM-8 系列**（即將量產）：48 TOPS 算力，功耗僅為同類方案的 **5%**

### IMC 的核心優勢與侷限

| 維度 | IMC 優勢 | IMC 侷限 |
|------|---------|---------|
| 功耗 | 可穿戴級（< 1 mA） | 不適合大模型訓練 |
| 算力密度 | 同工藝下提升 2 個數量級 | 模擬計算有雜訊，精度限 INT4–INT8 |
| 成本 | 極低 BOM | 工具鏈生態尚未成熟 |
| 容量 | 1–50 MB | 無法服務 100B+ 參數大模型 |
| 適用場景 | 邊緣端、IoT | 雲端推論目前力有未逮 |

## 四、兩條路線的完整對比

| 維度 | Groq LPU (SRAM) | 知存科技 IMC (Flash) | NVIDIA GPU (HBM) |
|------|----------------|---------------------|-----------------|
| 計算發生位置 | 獨立邏輯單元，權重在 SRAM | **記憶體陣列本身** | SM 核心，權重在 HBM |
| 記憶體類型 | 片上 SRAM（揮發性） | 非揮發性 Flash / ReRAM | 外部 HBM 堆疊 |
| 計算模式 | 數位、確定性 | **類比、統計性** | 數位、大規模並行 |
| 精度 | FP8–INT8 | INT4–INT8 | FP32 / BF16 / FP8 |
| 記憶體容量 | ~500 MB | 1–50 MB | 288 GB |
| 頻寬 | 150 TB/s | 極高（片內） | 22 TB/s |
| 延遲 | 超低、可預測 | 超低 | 批次相依 |
| 功耗 | 中低（~100–300W） | 極低（mA 等級） | 極高（700W+） |
| 模型規模 | 中型 LLM | 小型模型 | 1000億+ 參數 |
| 目標市場 | 雲端 AI 推論 | 邊緣 / IoT / 穿戴 | 訓練 + 雲端推論 |

## 五、市場賽道：根本不在競爭

這是理解這兩條路線最重要的一點——**它們目前不是競爭關係，而是分屬不同戰場**。

```
算力規模
    │
大型 ──  NVIDIA Rubin GPU + HBM
         ↓ 訓練 & 大批次推論
         ↓ 288GB / 22 TB/s / 700W+
         │
中型 ──  NVIDIA-Groq LPU + SRAM
         ↓ 雲端低延遲推論 / Agentic AI
         ↓ 500MB / 150 TB/s / ~200W
         │
邊緣 ──  知存科技 WTM IMC + Flash
         ↓ 穿戴 / AR / IoT / 助聽器
         ↓ 幾十 MB / 片內計算 / < 1mA
    │
小型
```

Groq 的被收購，本質上是 NVIDIA **向下延伸**到推論市場的防禦動作。知存科技的賽道則是 NVIDIA **尚未認真進攻**的超低功耗邊緣場景。

---

## 六、知存科技的未來賭注

NVIDIA 收購 Groq 其實**間接驗證了 IMC 的核心論點**：把記憶體和計算放得更近，是正確的方向。分歧只在於實現路徑和目標市場。

知存科技的長期勝負手在於：WTM-8 系列能否透過 **Chiplet 異質整合**，逐步爬升到中階推論市場（AR/VR、車載、智慧型手機），趕在 NVIDIA 和 Qualcomm 將功耗從上往下壓縮之前搶佔位置。

這場競賽的決定性視窗，大約在 **2026–2028 年**。

---

## 結語

後摩爾時代的算力競爭，早已不再是單純的電晶體縮微。記憶體牆正在催生出截然不同的架構創新：

- NVIDIA 用 200 億美元告訴市場：**推論時代來臨，低延遲才是關鍵**
- 知存科技用 mA 等級的功耗告訴市場：**邊緣 AI 的算力革命，剛剛開始**

兩條路，兩個市場，同一個方向——讓記憶體與算力，不再分離。
